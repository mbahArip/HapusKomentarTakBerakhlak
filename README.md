# Delete doodstrm and other

## Apa ini?
Fast dirty script yang bakal hapus semua komentar yang ngelink ke doodstrm.
script bakal jalan setiap 5 detik buat ngecek komentar yang ngelink ke doodstrm.

Domain akan diupdate secara berkala, kalau saia nemu akun kloningan ini pake domain lain.
Jika kalian nemu domain selain doodstrm dan komentar ga kehapus, bisa langsung kontak saia buat ditambahin.

NOTE: doodstrm **BUKAN** doodstream.
doodstrm dibuat sama seseorang cuman buat ngeredirect linknya ke telegram.

## Kenapa?
Lama-lama muak setiap ngeliat postingan di fb yang rame pasti muncul nih kloningan2 tak berakhlak ngeshare link bokep.
Dah itu aja alesan script ini dibuat.

## Cara install
1. Install tampermonkey, violentmonkey, atau greasemonkey. (tampermonkey sangat disarankan)
2. Install [script](https://raw.githubusercontent.com/mbaharip/HapusKomentarTakBerakhlak/main/src/deleteComment.user.js).
3. Refresh facebook.
4. Enjoy.

Jika menemukan bug / masalah / apapun bisa langsung komentar di post, bikin issue, pm lewat facebook, atau apapun dah.
